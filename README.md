# PocPosPaymentClient
Simple PocOpenPaymentClient

> Add it in your root build.gradle at the end of repositories:
...gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  .....
  > Step 2. add the dependency
... gradle 
	dependencies {
	        implementation 'com.github.DenisHDG:PocPosPaymentClient:Tag'
	}
  .....
